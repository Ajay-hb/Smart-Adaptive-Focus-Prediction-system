## 📖 Overview

The **Smart Adaptive Focus Prediction System** is a production-ready machine learning application that predicts a user's **focus score (0–100)** based on lifestyle and behavioral factors.

It also provides **actionable improvement suggestions**, helping users optimize productivity and focus levels.

---

## ✨ Key Features

| Feature | Description |
|--------|------------|
| 🔮 Focus Prediction | Predicts focus score (0–100) |
| 📊 Interactive Dashboard | Real-time predictions using Streamlit |
| 📈 Model Insights | Feature importance & analysis |
| 🔍 Behavior Analysis | Understand impact of lifestyle factors |
| 📊 Improvement Strategy | Suggests areas to improve focus |

| Mood | Emotional state |

## 📈 Insights

- Higher stress significantly reduces focus  
- Balanced sleep improves productivity  
- Excess screen time negatively impacts focus  
- Exercise improves cognitive performance
